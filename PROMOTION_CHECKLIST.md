# YapiYapi Fun Promotion Checklist

Use this checklist before spending real promotion budget.

## Current Position

- Site has 2 playable games: Space Adventure and Solar System Doodle.
- Good for free sharing and small family tests.
- Not ready for large paid acquisition until retention and ad compliance are proven.

## Small Test Plan

1. Share with 20-50 parents or teachers.
2. Ask whether children can start a game without help.
3. Watch whether children play at least 3 minutes.
4. Record which game they choose first.
5. Record where they stop or get confused.

## Metrics To Add Later

- Page views for home and each game page.
- Start button clicks.
- Level completion for Space Adventure.
- Mission completion for Solar System Doodle.
- Average session length.
- Return visits.

Do not collect child names, emails, photos, precise location, chat text, or profile data.

## AdSense Before Launch

- Add real AdSense publisher ID only after account approval.
- Publish `ads.txt` only with the real publisher ID.
- Use child-directed and non-personalized ad treatment where required.
- Keep ads outside active gameplay.
- Review sensitive ad categories before enabling ads.
- Use a Google-certified CMP where required for EEA, UK, and Switzerland traffic.

## Content Roadmap

- Improve Space Adventure retention with clearer rewards and a level map.
- Add one more playable game before scaling paid traffic.
- Add parent-facing screenshots or a short description for each game.
- Keep every public page in English for the initial US/EU audience.
